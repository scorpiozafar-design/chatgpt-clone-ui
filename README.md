# ChatGPT Clone (Static UI)

## Objective
This project is a static ChatGPT interface created using HTML and CSS. It focuses only on the user interface and does not include chatbot functionality.

## Features
- Chat window
- User and bot message bubbles
- Text input field
- Send button (non-functional)
- Responsive design
- Clean and minimal interface

## Design Process
- Used Flexbox for layout.
- Added a header, chat area, and input section.
- Styled user and bot messages with different colors.
- Used media queries to make the layout responsive.

## Technologies Used
- HTML5
- CSS3

## How It Resembles ChatGPT
- Dark theme similar to ChatGPT.
- Conversation-style message layout.
- Bottom input area with Send button.
- Responsive interface suitable for desktop and mobile devices.
